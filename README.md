# budget-tracker-server

Server for [Budget Tracker App](https://github.com/bmai53/budget-tracker)

Hosted on Heroku

### Tools Used:
* ExpressJS (controller)
* PassportJS (authentication)
* PostgreSQL (database)
* KnexJS (query builder)
* ObjectionJS (ORM)
* Mocha/Chai (testing)
